  opzione, viene supposto un argomento di default «.» (la directory
  corrente). L'opzione -d fa sì che le directory vengano trattate come
  file normali.  Un file è elencabile se il suo nome non inizia per
  «.» o se è presente l'opzione -a.

  usata l'opzione -l, ciascun elenco è preceduto da una riga di
  riassunto che dà la dimensione totale di tutti i file dell'elenco,
  misurata in mezzi chilobyte (512 byte).
